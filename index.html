Featured Project

PartyPal

GitHub

PartyPal is an application where you can find a curated list of highly rated caterer and venue recommendations based on location for your next party. I worked on integrating the Yelp API into our application to render venues and caterers onto our application. I also learned GraphQL to request specific data from the Yelp API and recieve only data a user requested on our front-end. I also styled various components of our application using CSS, styled-components and by incorporating styling frameworks like Material UI and React Bootstrap. Lastly, I developed the like functionality so when a user likes a venue or caterer it can be saved onto their account and in our database.

Featured Project

Cuidate

GitHub

Cuidate, meaning to take care of oneself in Spanish is a web application that randomly generates self-care activities for a user to try based on their answer to a check-in page. I developed the APIs needed to fetch random self-care activities from the database using Express. I rendered the self-care activity on the client side using React and Redux to handle my state. In addition, I designed the web application components with CSS, Styled-Components, and React-Bootstrap. Currently, I am working on a journaling feature that allows logged-in users to jot down any stressors from their day and have them saved to their account so they can keep track of their mental health.

Technologies used include

  • Node.js
  • React.js
  • Redux.js
  • Express.js
  • Sequelize.js
  • JSON Web Token
  • React-Bootstrap
  • Bycrypt
  • Styled-components

Green Garden

GitHub

Green Garden is a mock e-commerce web application where users can browse and purchase plants. I developed the logged-in user experience. This included logging in users with JSON Web Token, designing their shopping experience from browsing items to purchasing items on the application and developing the CRUD operations needed for a cart such as viewing existing items in a cart, adding items to a cart, updating cart items, and deleting items from cart. In addition, I ensured user interface responded to any state changes using React and React Bootstrap.

Featured Project

Number Guessing Game

Number Guessing Game is a web application where a player gets five chances to guess the correct number between 1 - 100. I developed the functionality for a player to submit a guess, get hints to what the correct number is, and reset the game. I also show the player helpful messages after submitting a guess letting them know if they are getting warmer or colder to the correct answer.
