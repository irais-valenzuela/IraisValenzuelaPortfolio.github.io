Featured Project

Cuidate

Cuidate (which means take care in Spanish) is a web application that randomly generates self-care activities for a user to try based on their answer to a check-in page. I developed the APIs needed to fetch random self-care activities from the database using express. I rendered the self-care activity on the client side using React and Redux to handle my state. In addition, I designed the web application components with CSS, styled-Components, and react-bootstrap. I recently added a journaling feature that allows logged-in users to jot down any stressors from their day and have them saved to their account so they can keep track of their mental health. I used react with redux to manage state changes such as creating a journal entry. I used postgresSQL to securely store users journal entries, express to write my APIs to handle CRUD operations related to journaling like getting journal entries and creating a journal entry. Finally, I used styled-components and CSS to design the journal component.
