Featured Project

PartyPal

PartyPal is an application where you can find a curated list of highly rated caterer and venue recommendations based on location for your next party! I worked on integrating the Yelp API into our application to render venues and caterers onto our application. I also learned GraphQL to get specific data from the Yelp API to optimize the results we got back from our API calls. I also designed various views of our application incorporating styling frameworks like Material UI and React Bootstrap. Lastly, I developed the like functionality so when a user likes a venue or caterer it can be saved onto their account and in our database. PartyPal GitHub

Cuidate

Green Garden

Green Garden is a mock e-commerce web application where users can browse plants and purchase them. I was responsible for the logged-in user experience functionality, which included logging in users with JSON Web Token and designing their shopping experience from browsing items to purchasing items on the application. Specifically, I developed the CRUD operations needed for a cart such as viewing existing items in a cart, adding items to a cart, updating cart items, and deleting items from cart. Featured Project

Number Guessing Game

Number Guessing Game is a web application where a player gets five chances to guess the correct number between 1 - 100. I developed the functionality for a player to submit a guess, get hints to what the correct number is, and reset the game. I also show the player helpful messages after submitting a guess letting them know if they are getting warmer or colder to the correct answer.
